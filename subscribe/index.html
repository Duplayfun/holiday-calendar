📥 日历订阅指南

如何使用？

  1. 在任意国家假期页点击「📥 导入日历」按钮
  2. 浏览器会下载一个 .ics 文件
  3. 点击下载的文件，系统会自动打开日历应用
  4. 在弹出的对话框中确认添加即可

各平台详细说明

iPhone / iPad

下载 .ics 文件后 → 点击文件 → 系统日历自动打开 → 点「添加全部」→ 完成

Android

下载 .ics 文件 → 打开 Google 日历 → 设置 → 导入 → 选择文件 → 导入

Google Calendar (网页版)

下载 .ics 文件 → 打开 calendar.google.com → 设置(齿轮) → 导入和导出 → 选择文件 → 导入

Outlook Calendar

下载 .ics 文件 → 打开 Outlook → 文件 → 打开和导出 → 导入/导出 → 导入 iCalendar 文件

温馨提示

  • 每个 .ics 文件包含一个完整年份的假期数据
  • 假期事件设置了提前 15 小时的通知提醒
  • 假期日期以"全天事件"形式添加，不会占用具体时间段
  • 建议年初导入当年日历，年底更新下一年的
